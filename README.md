# Olokas

AI search visibility monitoring for small business websites. Coming soon.

This repo currently hosts the pre-launch landing page deployed to [olokas.com](https://olokas.com) via Vercel. When the SaaS is built, this same repo becomes the Next.js app, with the marketing page becoming the `/` route.

## Stack
- Static HTML for now (single `index.html`)
- Hosted on Vercel
- Email capture via Formspree
- Auto-deploys on push to `main`
